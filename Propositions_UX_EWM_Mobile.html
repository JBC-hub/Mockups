🎯 Principes UX prioritaires
7 axes d'amélioration applicables à tous les écrans de mobilité
📸

Scan-First

Le scan code-barres/QR est la première action proposée. La saisie clavier reste disponible mais discrète.

👆

Zones de tap XXL

Tous les éléments interactifs ≥ 44px de hauteur. Pensé pour les gants d'entrepôt.

🎨

Code couleur strict

Rouge = urgent/retard · Orange = attention · Vert = OK/validé · Bleu = action en cours

⚡

Défaut intelligent

Pré-remplissage automatique de la quantité totale, du bin proposé et du filtre "Aujourd'hui".

✅

Feedback immédiat

Confirmation visuelle + son après chaque scan réussi. Message d'erreur explicite si problème.

🧭

Navigation simplifiée

Indicateur de progression (étapes 1/2/3…) visible en permanence. Retour arrière toujours accessible.

📊

Info contextuelle

Afficher uniquement les données utiles à l'écran courant. Masquer par défaut les champs rarement utilisés.

🔍 Synthèse par application
Points de friction identifiés et gains attendus

📦 GR Cockpit

  • Filtre "Aujourd'hui" actif par défaut → -2 taps pour démarrer
  • Progression packing visible sur la carte livraison (% barre)
  • Scan HU direct depuis la liste pour accéder instantanément au bon article
  • Bouton "GR tout" visible sans aller dans le menu
  • Écran packing : quantité totale pré-saisie, validation en 1 tap

🚚 Goods Issue Cockpit

  • Onglets Picking / Consolidation / GI avec badges compteurs
  • Création de tâche en 1 tap depuis la liste de picking
  • Confirmation destination bin : scan suggéré en grand, texte en petit
  • Alerte visuelle si bin de destination différent de celui proposé
  • Statut livraison en code couleur dès la liste

↔️ Bin to Bin

  • Mémorisation du dernier mode utilisé (HU vs produit) → 0 re-sélection
  • Sélection du Process Type réduite aux 3 les plus utilisés + "Autres"
  • Quantité : bouton "Tout" proéminent pour les cas 100% (majorité)
  • Confirmation immédiate si le Process Type le permet (sans écran supplémentaire)

🔢 Inventaire

  • Scanner le bin directement depuis la liste → saut automatique au document
  • Écran de comptage : clavier numérique grand format centré
  • Comparaison quantité attendue vs saisie en temps réel
  • Validation en 1 tap si quantité = stock théorique (cas fréquent)
  • Historique rapide des bins déjà comptés sur l'écran de liste
